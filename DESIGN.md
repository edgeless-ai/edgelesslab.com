# Edgeless Labs Design System

## Brand
- Name: Edgeless Labs
- Domain: edgelesslab.com
- Tagline: Creative technology lab. Tools for the AI-native developer.
- Tone: Direct, technical, opinionated, short, specific. No hype words.
- Audience: Developers and technical creators who use Claude Code, MCP servers, AI coding tools

## Colors

### Base (Light Theme)
- Page background: #F0F2F4 (cool neutral gray)
- Card surface: #FAFBFC
- Elevated: #FFFFFF (modals, dropdowns)
- Border subtle: #E2E5E9
- Border focus: #CBD0D6

### Text
- Primary: #1A1E24 (headings, body)
- Secondary: #6B7580 (descriptions, captions)
- Tertiary: #9CA3AD (placeholders)

### Accent - Fresh Mint (Primary)
- Main: #0D9668 (CTAs, links, highlights)
- Hover: #0BAD7A
- Muted: rgba(13, 150, 104, 0.10)

### Accent - Blue (Secondary)
- Main: #2563EB (interactive elements)
- Hover: #3B82F6

## Typography
- Headings: Inter (geometric, clean, technical)
- Body: System stack (-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif)
- Code: "SF Mono", "Fira Code", monospace
- Scale: 14/16/18/24/32/48/64px
- Line height: 1.5 body, 1.2 headings
- Weight: 400 body, 500 subheads, 600 headings, 700 hero

## Layout
- Max width: 1200px centered
- Grid: 12 columns, 24px gap
- Section padding: 80px vertical (desktop), 48px (mobile)
- Card border-radius: 12px
- Consistent 8px spacing grid

## Motion
- Duration: 200ms micro, 400ms section transitions
- Easing: cubic-bezier(0.4, 0, 0.2, 1)
- Scroll animations: Intersection Observer triggered, stagger 80ms between items
- Hero: sticky for first 100vh, ambient glow pulse

## Principles
- Tufte: Data density, no chartjunk, show the data
- Bauhaus: Form follows function, grid-based, no decoration
- Japanese: Negative space as design element, asymmetric balance
- Lab aesthetic: clean, precise, experimental but not chaotic
