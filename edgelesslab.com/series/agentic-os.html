## About This Series

Agentic OS documents the real-world engineering behind building an autonomous AI workforce. Each episode covers a distinct phase — from initial architectural reasoning and model routing decisions, through knowledge pipeline construction, to security hardening and self-healing infrastructure. This is not a theoretical exercise: every system described is live, running, and generating output. The architecture handles multi-agent Discord coordination, RSS/YouTube content triage, document processing, and trading — all operating autonomously with human-in-the-loop oversight.

Architecture & Cost

Your $12/Month AI Team That Works While You Sleep

The economics of autonomous AI infrastructure. How to build a multi-agent system on Hetzner VPS hosting for $12/month — model routing, cost optimization, and the architectural decisions that make it feasible. Covers the 8-bot swarm topology, Discord gateway collision patterns, and why open-weight models beat API calls at scale.

Model Routing

Intelligent Multi-Model Routing

The routing layer that directs queries to the right model — Kimi K2.5 for reasoning, Qwen for auxiliary tasks, specialized models for vision and compression. Covers the adapter pattern, provider fallback chains, and why routing quality matters more than individual model quality in production systems.

Knowledge Architecture

The Knowledge Base Loop

How autonomous agents research, enrich, and cross-reference knowledge across a 12,000+ file vault. The enrichment pipeline: RSS triage scoring, YouTube transcript clustering, NotebookLM synthesis, ChromaDB embeddings. Covers the KB score system (0-15 scale), auto-promotion thresholds, and how agents avoid confabulation through cross-referencing.

System Communication

File-Based Inter-Agent Communication

Why we replaced HTTP API calls with filesystem-based messaging for agent coordination. The queue JSONL pattern, atomic file claims, and how shared directories become message buses. Covers inbox dispatching, content-based routing with grep pattern matching, and why this architecture is more resilient than microservices for single-host multi-agent systems.

Security Deep Dive

Session Poisoning & Prompt Injection

How prompt injection attacks work on agent systems, why they're harder to defend against than traditional exploits, and the specific hardening measures we deployed: tiered skill loading, redaction filters, fail-closed security defaults, and the Tirith security framework. Real examples of injection attempts and the defense layers that caught them.

Creative Engineering

Real-Time Reaction Diffusion via WebGPU Compute Shaders

Pushing the visual frontier: building a biological simulation in the browser using WebGPU compute shaders. The reaction-diffusion algorithm, tile-based texture optimization, and how creative output validates that autonomous research produces genuinely novel results, not just assembled documentation.
