# Changelog

## v1.0.0 - 2026-04-16

Initial release.

### Generators
- Flow field generator (Perlin noise, SVG output)
- Reaction-diffusion simulator (Gray-Scott model, contour extraction)
- Circle packing with collision detection
- Moire interference pattern generator
- L-system renderer with 5 presets (Koch, Sierpinski, plant, dragon, Hilbert)
- Recursive subdivision with hatching and dot fill patterns

### Scoring
- Automated SVG scoring pipeline (ink coverage, line complexity, composition balance, visual entropy, plot feasibility)

### Guide
- Full guide covering flow fields, reaction-diffusion, geometric patterns, L-systems, moire/op art, scoring, pen plotter workflow, and parameter tuning

### Examples
- 5 flow field parameter presets
- 4 reaction-diffusion parameter presets
- Batch generation shell script
