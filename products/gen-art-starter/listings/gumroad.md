## Full Description

### The pain
Generative art tutorials show you how to draw one circle. They don't show you how to produce 1,000 variations and find the 10 worth printing. The gap between "interesting algorithm" and "art on paper" is parameter tuning, output scoring, and plotter workflow. That's where the real work lives.

### The credential
This kit comes from an art practice that has generated 16,000+ scored pieces across 17 algorithm families, with a 5-signal automated scoring rubric that filters output down to the top 2%. The pen-plotter editorial at edgelesslab.com/pen-plotter showcases 1,296 curated pieces from this system.

### What's inside
- **6 production generators** — Python scripts that output print-ready SVG files. Each is well-commented with parameter explanations.
- **Automated scoring rubric** — a 5-signal scoring script that rates any SVG on a 0-100 scale
- **The full guide** (~5,500 words) — algorithm deep dives, parameter tuning methodology, pen plotter workflow, scoring philosophy
- **Parameter presets** — JSON configs for each generator with named presets (calm, turbulent, dense, etc.)
- **Batch generation script** — run all generators with defaults in one command

### The 6 generators
1. **Flow Field** — Perlin noise-driven particle traces. The most versatile generator family. Calm rivers to turbulent storms based on noise parameters.
2. **Reaction-Diffusion** — Gray-Scott model simulation. Produces organic coral, spots, stripes, and maze patterns from two interacting chemicals.
3. **Circle Packing** — Fills space with non-overlapping circles. Variable radii, collision detection. Clean geometric compositions.
4. **Moiré Patterns** — Overlapping rotated grids that create interference patterns. Op art inspired. Prints beautifully at any scale.
5. **L-Systems** — Formal grammars rendered with turtle graphics. 5 presets included: Koch curve, Sierpinski triangle, botanical plant, dragon curve, Hilbert space-filling curve.
6. **Recursive Subdivision** — Splits rectangles with random cuts, applies fill patterns. Mondrian meets generative randomness.

### The scoring rubric (5 signals)
- **Ink coverage** — is there enough on the page? (filters blank/sparse failures)
- **Line complexity** — path count, curve ratio, total geometry
- **Composition balance** — rule-of-thirds focal point distribution
- **Visual entropy** — edge variance (too uniform = boring, too chaotic = noise)
- **Plot feasibility** — can a pen plotter actually draw this without jamming?
