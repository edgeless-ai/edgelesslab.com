FAQ

Does this work on Windows or Linux?
The setup script is optimized for macOS 14+ (Apple Silicon). Linux instructions are included but not automated. Windows support is not included - we recommend WSL2 or a Linux VM.
Do I need API credits to use the agents?
You need API keys for at least one LLM provider (OpenAI, Fireworks, etc.). The kit includes provider fallback configs so you can switch if one hits rate limits. We recommend starting with Fireworks for cost efficiency.
Is this an official NousResearch product?
No. This is built by Edgeless Labs, an independent team that runs Hermes in production daily. We are active contributors to the Hermes ecosystem.
What if I get stuck during setup?
The kit includes Discord community access. Post in the #agent-setup channel and get help from people who run this infrastructure daily. Response time is typically under 4 hours.
