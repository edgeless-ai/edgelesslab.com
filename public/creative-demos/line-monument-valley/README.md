# Line-Monument Valley

A single-file, self-contained WebGL demo built with Three.js.

## Concept

Scroll through a dark 3D valley where each letter of the alphabet is a monumental wireframe sculpture drawn by an invisible algorithmic pen-plotter. One monument — the letter **M** — is a glowing anomaly that glitches the camera.

This demo fuses two creative techniques:

- **Active Theory-style immersive camera choreography:** Scroll position maps to a parameter `t` along a Catmull-Rom spline path. The camera is locked to the path, creating a cinematic valley fly-through. Mouse movement adds a subtle parallax gaze offset.
- **Inconvergent-style algorithmic line art:** Each letter is a deterministic constrained random walk (1200 steps, 8 directions, bounded to a 100×100 box) extruded into 3D by step index. The result is a letter-shaped mountain of single-weight lines.

## Technique Notes

- **Camera path:** `THREE.CatmullRomCurve3` with 26 points, `tension = 0.5`.
- **Scroll smoothing:** `targetT` updated on `wheel`; `currentT` lerps to it with factor `0.05`.
- **Random walk:** Seeded LCG pseudo-random generator per letter. 8 directions at 45° increments, step size 2, bounce off walls.
- **Extrusion:** Z-depth = `stepIndex * 0.1`, creating a rising terrain effect.
- **Anomaly:** `IcosahedronGeometry(40, 1)` with `MeshStandardMaterial` (`emissive: #00FFFF`). Includes a `PointLight` with distance 300.
- **Camera stutter:** Simple value-noise jitter (±2 units on X/Y) when `t` is within 0.1 of the anomaly.
- **State machine:** Overlay opacity ramps by proximity within each monument segment (IDLE → REVEALING → ACTIVE → FADING).
- **Palette:** Cinematic noir — `#000000` void, `#FFFFFF` wireframe, `#00FFFF` anomaly glow, `#E8A838` headings, `#F0F0F0` body, `#16213E` fog.
- **Controls:** Scroll to travel, Mouse to gaze, **R** to reset, **A** to toggle anomaly bloom intensity.

## Running

Open `index.html` directly in any modern browser. No build step, no external assets (Three.js is loaded from CDN).
