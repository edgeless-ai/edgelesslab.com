1B. Mass Balance Around Fryer Perry's Ch. 2, §2-1

Four streams cross the fryer boundary: wet chips in, finished chips out, water evaporated, and oil absorbed (net negative leaving fryer inventory). Dry corn solids are conserved; water and oil are not.

// Step 1 — Dry solids out (no oil yet counted in this stream)
ṁsolids,out = ṁout × (1 − xw,out) = 150 × (1 − 0.015) = 147.75 lb/hr

// Step 2 — Oil absorbed into chips
ṁoil,abs = ṁout × xoil = 150 × 0.30 = 45 lb/hr

// Step 3 — Dry corn solids (solids minus absorbed oil)
ṁcorn,dry = ṁsolids,out − ṁoil,abs = 147.75 − 45 = 102.75 lb/hr

// Step 4 — Wet chip feed rate (masa + water entering)
ṁin = ṁcorn,dry / (1 − xw,in) = 102.75 / (1 − 0.524) = 215.9 lb/hr

// Step 5 — Water carried in with chips
ṁH2O,in = ṁin − ṁcorn,dry = 215.9 − 102.75 = 113.1 lb/hr

// Step 6 — Water retained in finished chips
ṁH2O,out = ṁout × xw,out = 150 × 0.015 = 2.25 lb/hr

// Step 7 — Water evaporated (latent load driver)
ṁevap = ṁH2O,in − ṁH2O,out = 113.1 − 2.25 = 110.9 lb/hr

// Step 8 — Oil makeup (replace absorbed oil)
ṁoil,makeup = ṁoil,abs = 45 lb/hr = 45 / 7.5 = 6.0 gal/hr

1C. Heat Duty Calculation Perry's Ch. 11, §11-4

Four heat loads are summed: sensible heating of wet chips, latent heat of water evaporation, heat loss replacing cool makeup oil, and radiation/convection losses from the fryer shell. The latent term dominates (>68% of process duty).

// Q_sensible — bring chips from 75°F to frying temp
Qsens = ṁin × Cp,chip × (Tfry − Tin)
Qsens = 215.9 × 0.45 × (357 − 75) = 215.9 × 0.45 × 282
Qsens = 27,431 BTU/hr

// Q_latent — evaporate water at atmospheric boiling point
// λ at 212°F = 970.3 BTU/lb (steam tables, Perry's App. A)
Qlat = ṁevap × λ = 110.9 × 970.3
Qlat = 107,608 BTU/hr

// Q_oil — heat makeup oil from 70°F ambient to 357°F frying temp
Qoil = ṁoil,makeup × Cp,oil × (Tfry − Tambient)
Qoil = 45 × 0.47 × (357 − 70)
Qoil = 6,068 BTU/hr ≈ estimated 6,100 BTU/hr

// Q_rad — radiation + convection from uninsulated fryer shell
// Estimated at ~5% of process heat (Perry's §11-14; unlagged equipment)
Qrad ≈ 0.05 × (Qsens + Qlat) = 0.05 × 135,039 = 6,752 BTU/hr → use 8,000 BTU/hr

// Q_total — sum of all loads
Qtotal = Qsens + Qlat + Qoil + Qrad
Qtotal = 27,431 + 107,608 + 6,100 + 8,000
Qtotal = 149,139 BTU/hr

// Q_fuel — burner duty at 80% thermal efficiency
Qfuel = Qtotal / ηthermal = 149,139 / 0.80
Qfuel = 186,424 BTU/hr

// Natural gas: HHV = 1,020 BTU/SCF (standard, Perry's App. A)
V̇NG = Qfuel / HHVNG = 186,424 / 1,020
V̇NG = 182.8 SCF/hr → specify 185 SCF/hr (round up)

1E. Heat Transfer Coefficient & Area Check Perry's Ch. 11, §11-10

Submerged frying is a complex combination of forced convection (belt movement + bubble agitation) and nucleate boiling from the chip surface as pore moisture flashes. Perry's cites overall U for immersion fryers in the range 250–300 BTU/(hr·ft²·°F). We use U = 275 as a midpoint estimate.

// Overall heat transfer coefficient (submerged frying, nucleate boiling regime)
U = 275 BTU/(hr·ft²·°F) [Perry's: 250–300 range]

// Driving force: oil at 357°F, chip surface at ~212°F (water boiling)
ΔT = Tfry − Tboil = 357 − 212 = 145 °F

// Required heat transfer area for latent load only (rate-limiting step)
Areq = Qlat / (U × ΔT) = 107,608 / (275 × 145)
Areq = 2.70 ft²

// Actual belt area in fry zone
Abelt = W × Lfry = 2 × 6 = 12 ft²
// Ratio: 12 / 2.70 = 4.4× → fryer is area-generous; belt sizing governs, not heat transfer
Area check: OK — heat transfer area is not the constraint at this scale.

2B. Mixing Power — Newton's Power Number Correlation Perry's Ch. 18, §18-4

For high-viscosity doughs in the laminar regime (Re ≪ 10), the power number Np is inversely proportional to Re. For planetary/anchor agitators mixing stiff dough: Np · Re ≈ Kp, with Kp ≈ 70 at Re = 1 (Perry's Table 18-2, anchor impeller, laminar). The power equation is then:

// Newton's power correlation (general form)
P = Np × ρ × N³ × D⁵
// where P = power [W], ρ = fluid density [kg/m³], N = rotation rate [rev/s], D = impeller dia [m]

// Given (SI units for this calculation)
Np = 70 [anchor/planetary, laminar, Perry's Table 18-2]
ρdough = 75 lb/ft³ × 16.018 = 1,201 kg/m³
N = 60 RPM = 1.0 rev/s
D = 14 in bowl → impeller sweep ≈ 12 in = 0.305 m; D = 0.35 m (planetary orbit dia)

// Compute D⁵
D⁵ = 0.35⁵ = 0.35² × 0.35² × 0.35 = 0.1225 × 0.1225 × 0.35 = 5.25 × 10⁻³ m⁵

// Power
P = Np × ρ × N³ × D⁵
P = 70 × 1,201 × 1.0³ × 5.25×10⁻³
P = 441 W = 0.59 HP

// Standard 60 qt commercial mixer: 1.5–2 HP motor
// Safety factor = 1.5 HP / 0.59 HP = 2.5× → adequate for startup torque and dough variability
Motor rating: OK. 60 qt / 1.5 HP planetary mixer meets requirements with 2.5× safety factor.

3A. Biot Number Check & Lumped Capacitance Analysis Perry's Ch. 5, §5-7

Lumped capacitance (well-mixed body) is valid when Bi = hL/k < 0.1. For a thin chip, the characteristic length L = V/As ≈ half-thickness. We check this criterion before applying the exponential decay model.

Ttarget115°FTarget chip temp for bagging

// Biot number check
Bi = (h × Lc) / kchip = (5 × 0.005) / 0.15
Bi = 0.167
// Bi = 0.167 > 0.1 — borderline. Lumped capacitance will slightly underestimate cooling time.
// Acceptable for sizing (conservative direction: will oversize conveyor slightly).

// Time constant τ = ρ Cp V / (h A)
// For a flat chip (two-sided cooling): V/A = L_c = 0.005 ft
τ = (ρchip × Cp,chip × Lc) / h
τ = (75 × 0.40 × 0.005) / 5
τ = 0.030 hr = 1.8 min

// Dimensionless temp at target: θ = (T_target - T_∞) / (T_in - T_∞)
θ = (115 − 70) / (350 − 70) = 45 / 280 = 0.161

// Lumped capacitance model: θ = exp(-t/τ) → t = -τ ln(θ)
tsingle = −τ × ln(θ) = −0.030 × ln(0.161) = −0.030 × (−1.826)
tsingle = 0.0548 hr = 3.3 min

// Bed correction: chip pile impedes air circulation. Apply 1.5× factor for shallow bed.
tbed = tsingle × fbed = 3.3 × 1.5
tbed ≈ 5.0 min (use 5 min design cooling time)
