# Task: Build Texture and Hatching Library

## Priority: Medium
## Status: Not Started
## Category: Artistic Tools

## Description
Develop a comprehensive library of textures and hatching patterns optimized for pen plotting, with tools to apply them to shapes and regions.

## Requirements
1. **Basic Patterns**
   - Parallel lines (various angles)
   - Cross-hatching (single and double)
   - Dots/stippling
   - Scribbles
   - Waves and curves
   - Geometric patterns

2. **Advanced Textures**
   - Wood grain
   - Fabric/cloth
   - Stone/marble
   - Water ripples
   - Grass/fur
   - Scales/feathers

3. **Application Tools**
   - Fill detection for closed shapes
   - Gradient hatching (density variation)
   - Directional hatching following contours
   - Texture mixing and layering

4. **GUI Controls**
   - Pattern selector with previews
   - Density/spacing controls
   - Angle and variation
   - Randomness factors
   - Clipping to shapes

## Technical Specifications
- Efficient line generation algorithms
- SVG pattern optimization
- Mask/clipping support
- Pattern tiling system

## Integration
- Apply to cellular automata regions
- Fill reaction-diffusion contours
- Shade 3D-like forms
- Background textures for compositions
