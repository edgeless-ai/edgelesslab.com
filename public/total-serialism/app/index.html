Welcome to Total Serialism

A collection of algorithms for creating generative art optimized for pen plotters.

Getting Started

  1. Browse - Use the category filters or search to find an algorithm
  2. Experiment - Adjust parameters using the controls panel
  3. Regenerate - Click "Regenerate" or use a new seed for variations
  4. Export - Download as SVG for pen plotting or PNG for sharing

Key Features

  • Shareable URLs - Copy your exact configuration to share with others
  • Seed Control - Use the same seed to reproduce exact outputs
  • SVG Export - Vector output optimized for pen plotters
  • Paper Sizes - Preset sizes for A4, A3, Letter, and custom dimensions

For Pen Plotter Users

  • SVGs use stroke-only rendering (no fills) for clean plotting
  • Path optimization reduces pen travel distance
  • Use the Plotter Preview tool to simulate drawing
  • Layer export available for multi-color plotting
